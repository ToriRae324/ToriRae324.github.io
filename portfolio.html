Bamazon

Bamazon

Bamazon is a CLI app used in conjunction with MySQL that runs as a simple storefront. The app displays products from the database, takes in orders from customers and depletes stock from the store's inventory The app allows Managers and Supervisors to perform additional functions.

In order to run this program node modules will need to be installed from the package.json file, and the database will need to be created.

LIRI

LIRI

LIRI is a command line node app that takes in specific parameters from the command line and gives you back data in the terminal/bash window.

In order to run this program node modules will need to be installed from the package.json file, and a .env file with Twitter and Spotify keys will need to be added.
